### Fuzzscan是一款信息收集工具，支持对目标资产网段的端口扫描与web服务的探测。
此工具仅用于授权情况下的攻防演习资产信息收集




### 用法简单usage:

python Fuzzscan.py -r 1.1.1.1/24

python Fuzzscan.py -f 1.txt
